ผลงาน (Portfolio)

เว็บไซต์ Portfolio

เว็บไซต์ Portfolio ที่พัฒนาด้วย HTML และ CSS เพื่อใช้รวบรวมประวัติส่วนตัว ผลงาน และข้อมูลการศึกษา

เครื่องมือที่ใช้ : HTML, CSS

เว็บไซต์ฝึกเขียน HTML

เว็บไซต์ที่สร้างขึ้นเพื่อฝึกการใช้ HTML และการจัดวางโครงสร้างของหน้าเว็บ

เครื่องมือที่ใช้ : HTML

เว็บไซต์ตกแต่งด้วย CSS

ฝึกการใช้ CSS ในการตกแต่งเว็บไซต์ เช่น สี พื้นหลัง ตัวอักษร และการจัดวางเนื้อหา

เครื่องมือที่ใช้ : HTML, CSS


ทักษะที่ได้รับ

  • สามารถสร้างเว็บไซต์ด้วย HTML
  • ตกแต่งเว็บไซต์ด้วย CSS
  • จัดโครงสร้างหน้าเว็บให้เหมาะสม
  • อัปโหลดเว็บไซต์ขึ้น GitHub Pages
